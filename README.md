## Overview
This repository is designed as a structured operations training system for a new Systems Administrator role. It combines study guides, implementation notes, scripts, and labs so you can build confidence through both reading and hands-on practice.

### Included domains
- Red Hat Linux administration
- Windows and Linux mixed infrastructure support
- Azure infrastructure operations
- Terraform infrastructure as code
- STIG hardening and compliance
- Server upgrades and patching
- Vulnerability remediation
- Single-VM cyber range practice labs

## Why this project exists
The goal is to provide a practical, expandable knowledge base that can evolve into a real operational handbook for production-style environments.
